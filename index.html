    🔧 Set Map Center

    Pan and zoom to center the map as you want it. Then use the "Copy center & zoom" button to copy that information so you can paste it into map-settings.js. Use Copy center & zoom, paste into MAP_CENTER_INPUTS.center and MAP_CENTER_INPUTS.zoom, commit in GitHub, then refresh this page.

    When your center is final, set MAP_CENTER_INPUTS.showCenterPanel to false in map-settings.js.

    🔧 Layer Toggle Field

    Setup mode: choose a field to group toggle checkboxes (for example: category, district, type).

    Pick a field, click Copy toggle field line, paste it into LAYER_TOGGLE_INPUTS in map-settings.js, commit in GitHub, then refresh this page.

    If you do not want grouped toggles, use None (toggle each feature) and keep toggleFieldKey: "". When done, set LAYER_TOGGLE_INPUTS.showLayerTogglePanel to false.

    🔧 Available Fields

    Setup mode: this list shows every property detected in your GeoJSON.

    Click Copy popup line, paste lines into POPUP_DATA_INPUTS.popupSections in map-settings.js, commit in GitHub, then refresh this page.

    When your popup fields are finalized, set POPUP_DATA_INPUTS.showFieldPanel to false in map-settings.js.

    🔧 Color Settings

    Setup mode: pick a scheme below — the map updates live. When done, set COLOR_INPUTS.colorScheme to your chosen name in map-settings.js and set showColorPanel: false.

    To use your own colors, choose Custom and edit the customScheme block in map-settings.js.
