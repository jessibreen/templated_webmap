Webmap Template

Load a GeoJSON file from the data folder and customize popups in map-settings.js.

Set Map Center

Setup mode: keep this published map page and map-settings.js open in two tabs. Pan/zoom here, then paste values there.

Use Copy center & zoom, paste into MAP_CENTER_INPUTS.center and MAP_CENTER_INPUTS.zoom, commit in GitHub, then refresh this page.

When your center is final, set MAP_CENTER_INPUTS.showCenterPanel to false in map-settings.js.

Layer Toggle Field

Setup mode: choose a field to group toggle checkboxes (for example: category, district, type).

Pick a field, click Copy toggle field line, paste it into LAYER_TOGGLE_INPUTS in map-settings.js, commit in GitHub, then refresh this page.

If you do not want grouped toggles, use None (toggle each feature) and keep toggleFieldKey: "".

Available Fields

Setup mode: this list shows every property detected in your GeoJSON.

Click Copy popup line, paste lines into POPUP_DATA_INPUTS.popupSections in map-settings.js, commit in GitHub, then refresh this page.

When your popup fields are finalized, set POPUP_DATA_INPUTS.showFieldPanel to false in map-settings.js.
